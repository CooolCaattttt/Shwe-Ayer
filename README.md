# Shwe-Ayer
This project is E-commerce project.
1. Download the Zip File
2. Download the Xampp Application from https://www.apachefriends.org/download.html
3. Make installation setup for Xampp
4. Extract the Zip File under C : / xampp / htdocs
5. Open Xampp Application and run Apache and MySQL (Both must turn to green if you see red you need to fix it)
6. Open Browswer and type ( localhost/your project name/index.html )
7. For Admin login ( localhost/your project name/login.php )
